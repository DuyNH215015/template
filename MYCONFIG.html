Hướng dẫn sử dụng file cấu hình IoT

Đây là hướng dẫn giúp bạn tạo và import file cấu hình IoT. File JSON này định nghĩa các thông số của thiết bị, danh sách thành phần (component) như text box, nút bấm, dropdown để tương tác với hệ thống IoT qua giao diện Web Serial.

Các bước sử dụng

  1. Bấm nút tải template bên dưới để tải file mẫu.
  2. Mở file JSON và chỉnh sửa các tham số theo nhu cầu.
  3. Import file JSON vào giao diện quản lý để nạp cấu hình cho thiết bị.
